The Road Ahead

Future States

What we’re building, what we’re reaching for, and what we dream about

The Universality Thesis

The fuse is engineering. The fire is physics. Any codec that uses a partition → transform → quantize pipeline is susceptible. The mechanism is universal.

Images

JPEG, WebP, HEIF — all use block-based DCT or wavelet quantization. Tested and confirmed across JPEG and WebP with cross-codec transcoding.

Audio

MP3, AAC, Opus — frequency-domain quantization in sub-band partitions. Initial MP3 testing has exceeded expectations.

Video

H.264, H.265, AV1 — block-based motion-compensated DCT quantization. The same thermodynamic tax applies per-frame.

I

Must Build

The work we believe in and are committed to delivering

CLI Detection Tool

A single binary that takes a file and returns a detection state (A/B/C/D) with confidence scores. No cloud. No API key. No dependencies. The file walks in, the measurement walks out. This is the table stakes — if detection requires infrastructure, we’ve already lost the argument.

Audio Provenance (MP3/AAC)

The MDCT frames in audio compression create the same boundary structure as DCT blocks in images. Initial MP3 testing exceeded expectations. Full audio pipeline — embed, cascade, detect — validated across MP3, AAC, and Opus. I-frames are JPEGs in a trench coat. Audio frames are the same thing with a different hat.

Payload Encoder v1

Layer F is architecturally ready but not yet validated at scale. Structured flanking data — creator ID, timestamp, section index — encoded into the position pattern. The fingerprint becomes a message. The message survives what the file survives.

DIV2K+ Validation Suite

Expand from 800 to 5,000+ images across diverse content classes. Characterize per-class amplification behavior. Map the landscape of where granite is strongest and where sandstone fights back. Publish honest results, including the failures.

Lightroom / Photoshop Plugin

Provenance at the point of creation. One checkbox at export time. The photographer decides. Nobody else. If adoption requires changing workflows, adoption won’t happen. It has to live where the work already lives.

Open Specification Document

A formal, versioned specification for the embedding and detection protocol. Not a paper — a spec. Implementable by anyone. BSD-licensed. If only one implementation exists, it’s a product, not a protocol. We need the spec to exist independently of us.

II

Stretch Goals

Hard problems worth solving if the foundation holds

Video Provenance (H.264/H.265/AV1)

Every I-frame is a JPEG. Every P-frame and B-frame carries forward the variance anomaly through motion compensation. The thermodynamic tax compounds across the GOP. The challenge: embed once, detect across temporal compression that spans hundreds of frames. The physics says yes. The engineering is non-trivial.

Corpus-Scale Attribution Service

A matching service that compares fingerprints across a corpus. Not identity — similarity. “This file shares a provenance fingerprint with these other files.” Jaccard similarity at scale. The hard part isn’t the algorithm, it’s the trust model: who runs it, who queries it, what gets logged. Participation over permission extends to the infrastructure.

Cross-Platform Detection API

An open API that platforms can integrate voluntarily. Upload receives a detection state. No enforcement. No blocking. Just measurement. Platforms that want to surface provenance information can. Platforms that don’t, don’t. The measurement exists regardless.

Adversarial Robustness Testing

Red team the system with generative attacks. Style transfer. Inpainting. Partial reconstruction. Deep fake pipelines that touch every pixel. Find out where the signal actually breaks — not where we think it breaks, where it actually breaks. Publish everything.

Mobile Camera Integration

Provenance embedded at capture time on the device. Before the image touches a network. Before anyone else can claim it. The phone is the first witness. This requires hardware partnerships and OS-level integration — a stretch, but the right place for the signal to originate.

III

Moonshot

The ideas that sound unreasonable until they aren’t

Generative AI Provenance

Every image, audio clip, or video generated by an AI model carries a provenance signal from birth. Not a visible watermark. Not metadata that can be stripped. A statistical property baked into the generation process itself — the model’s DCT quantization choices become the embedding surface. The output is born marked. The question “was this AI-generated?” becomes a detection state, not a debate.

Universal Media Provenance Layer

A protocol-level layer that any media format can implement. Not a standard controlled by a committee — a thermodynamic fact that any codec must respect. Like TCP/IP for provenance: not owned, not permissioned, not optional for physics. Every lossy codec that partitions, transforms, and quantizes is already participating whether it knows it or not. Make that participation legible.

Provenance Archaeology

Retroactive detection on existing media. Billions of images already on the internet carry compression artifacts that encode their handling history. Nobody embedded a signal — but the codec’s own quantization decisions are a signal. The variance landscape of a five-times-compressed JPEG is a forensic record. Read the geology of digital media. Sandstone all the way down, and we know what the layers mean.

Real-Time Broadcast Provenance

Live video streams carrying provenance signal in real time. Every frame marked as it’s encoded. Detection at the receiving end, during playback, before anyone can re-encode or strip. The gap between capture and verification collapses to zero. Deepfakes in live broadcasts become detectable in the same stream they arrive in.

The Anti-Forgery Constant

Prove mathematically that there exists a lower bound on the energy required to remove the variance anomaly without destroying the content. Not just “it’s hard to remove” — it is thermodynamically impossible to remove without visible degradation. A proof that the cost of forgery exceeds the value of forgery. The granite isn’t just durable. It’s provably durable. Turn the conjecture into a theorem.

The fuse is what we build. The fire is what physics gives us for free.
